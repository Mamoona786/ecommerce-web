MongoDB Atlas instead of only local Compass for production

Vite provides standard production build/deploy guidance, and Compass is fine for local development, but deployment will need a hosted MongoDB URI rather than only a local database.

Important note about MongoDB Compass

Compass is mainly a GUI tool to view and manage MongoDB data.
Your app will connect to MongoDB through the connection string in Mongoose, not directly “through Compass.” Mongoose handles the app connection, while Compass helps you inspect the database.

So for local development:

install MongoDB server

use Compass to inspect collections

use Mongoose in code to connect
